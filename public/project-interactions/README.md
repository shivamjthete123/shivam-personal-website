# Project Interaction Assets

Store visitor-facing project assets here.

Recommended pattern:

- `public/project-interactions/project-lens.html`
- `public/project-interactions/<project-slug>/demo.html`
- `public/project-interactions/<project-slug>/case-study.pdf`
- `public/project-interactions/<project-slug>/walkthrough.mp4`

Reference each asset from the matching file in `content/projects/`.
